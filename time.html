Two-Month Weekly Curriculum for Computer Skills Training

Week 1: Introduction to Computers and Basic Skills

Introduction to computers, their components, and how they work.
Basic computer operations: power on/off, restart, log in/out.
Navigating the desktop and using the mouse and keyboard effectively.
Introduction to file management: creating, renaming, and deleting files/folders.
Introduction to basic software applications: word processing, paint/drawing, and web browsing.

Week 2: Internet and Web Browsing

Introduction to the internet and its uses.
Understanding web browsers and their features.
Safe internet usage and online etiquette.
Searching the web effectively: using search engines, evaluating search results.
Introduction to bookmarking and saving web pages.

Week 3: Word Processing and Typing Skills

Introduction to word processing software (e.g., Microsoft Word, Google Docs).
Basic formatting techniques: font styles, sizes, and colors.
Creating, editing, and saving documents.
Typing exercises to improve speed and accuracy.
Introduction to basic formatting tools: bullets, numbering, and alignment.

Week 4: Digital Presentations

Introduction to presentation software (e.g., Microsoft PowerPoint, Google Slides).
Creating simple slideshows: adding text, images, and basic transitions.
Using design elements effectively: color schemes, backgrounds, and fonts.
Presenting skills: organizing content, using visuals, and speaking confidently.

Week 5: Spreadsheets and Data Management

Introduction to spreadsheets (e.g., Microsoft Excel, Google Sheets).
Understanding rows, columns, and cells.
Data entry and formatting: numbers, dates, and formulas.
Basic calculations and using built-in functions.
Sorting and filtering data.

Week 6: Cybersecurity and Online Safety

Understanding the importance of cybersecurity and online safety.
Safe internet practices: passwords, privacy settings, and online identity.
Recognizing online threats: phishing, malware, and scams.
Internet ethics and responsible online behavior.
Introduction to parental controls and digital footprints.

Week 7: Multimedia Presentations

Introduction to multimedia presentation tools (e.g., PowerPoint, Google Slides).
Adding images, audio, and video to presentations.
Using multimedia elements effectively: transitions, animations, and timings.
Creating engaging and interactive presentations.
Presenting multimedia presentations confidently.

Week 8: Data Analysis and Graphs

Utilizing spreadsheet software for data analysis (e.g., Excel, Google Sheets).
Creating charts and graphs to represent data
